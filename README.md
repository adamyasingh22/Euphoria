# 🛍️ Euphoria – Fashion E-commerce Store

Euphoria is a modern and fully responsive fashion e-commerce web application built using **Next.js**, **styled-components**, and **Redux Toolkit**. It features dynamic product pages, a shopping cart with live quantity updates, and secure cookie-based authentication using custom middleware.

## 🔧 Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** styled-components, Tailwind CSS
- **State Management:** Redux Toolkit
- **Authentication:** JWT, Cookie-based session management
- **Routing:** Next.js App Router
- **Middleware:** Custom middleware for route protection
- **Animations:** Framer Motion
- **API Integration:** Axios (RESTful APIs)
- **Deployment:** Vercel

## ✨ Features

- 🛒 Add to Cart & Update Quantity
- 🔐 Protected Routes with Middleware
- 🍪 Cookie-based Session Handling
- 📱 Fully Responsive UI (Mobile-first)
- 🎨 Styled with styled-components & Tailwind
- ⚙️ Modular Component Architecture
